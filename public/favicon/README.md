# Favicon Assets Checklist

Place all files from this list into `client/public/favicon/`.

## Browser / Desktop
- `favicon.ico` - ICO, contains `16x16`, `32x32`, `48x48`
- `favicon.svg` - SVG, vector icon
- `favicon-16x16.png` - PNG `16x16`
- `favicon-32x32.png` - PNG `32x32`
- `favicon-48x48.png` - PNG `48x48`

## Apple / iOS
- `apple-touch-icon-180x180.png` - PNG `180x180`

## Android / PWA
- `android-chrome-192x192.png` - PNG `192x192`
- `android-chrome-512x512.png` - PNG `512x512`
- `android-chrome-maskable-512x512.png` - PNG `512x512` (safe area for maskable icons)

## Windows tiles
- `mstile-150x150.png` - PNG `150x150`

## Social networks preview
- `og-image-1200x630.png` - PNG `1200x630`

## Already prepared config files in this folder
- `site.webmanifest`
- `browserconfig.xml`
